In this post I'll explain how I use [Formiddable's](https://twitter.com/FormidableLabs) excellent
[Prism React Renderer](https://github.com/FormidableLabs/prism-react-renderer) to add syntax highlighting to code blocks
in both MDX v1 and MDX v2 with the help from [gatsby-plugin-mdx](https://www.gatsbyjs.com/plugins/gatsby-plugin-mdx/).

## MDX v1

`gatsby-plugin-mdx`: `^3.0.0`

## MDX v4

`gatsby-plugin-mdx`: `^4.0.0`
