## Quick start

1. Install Node.js >= 18
2. Install Gatsby CLI >= 5
3. Install packages: `npm i`
4. Start site: `gatsby develop`
5. Locate the browser at http://localhsot:8000
6. Locate GraphQL http://localhost:8000/___graphql
